# Personal Budget

## A simple Excel spreadsheet to help control personal finance.

I decided to create this spreadsheet because I was having trouble paying my bills on time, and also the fact that I didn't know where my money was going always intrigued me.

## Features

This Personal Budget is composed of six different plans, each addressing different aspects:

1. Data Validation - Here you set the names of your categories.
2. Balance & Tracking - In this plan, you record all financial operations.
3. Net Salary - Contains all the values related to Income, Deductions, and Investments, divided by months.
4. Expenses - Contains all the values related to Expenses, categorized and divided by months.
5. Graphic Categories - Dynamic graphical representation of different categories.
6. Graphic Annual - Dashboard of all relevant information.

* Note: Keep Data Validation, Net Salary, Expenses, and Graphic Categories hidden for better navigation, and use them just for reference.
